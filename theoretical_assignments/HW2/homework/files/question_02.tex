\section{پرسش ۲.}
فرض کنید $\mathcal{H}$ فضای فرضیه درنظر گرفته شده برای یک مسئله یا \textbf{مفهوم}\LTRfootnote{Consept} است. از میان تمام فرضیه‌هایی که درون مجموعه $\mathcal{H}$ قرار دارند، کدام یک بهترین فرضیه برای یک مسئله است؟ آیا الزاماً این فرضیه دارای خطای تجربی\LTRfootnote{Empirical Error} صفر است؟ توضیح دهید.
